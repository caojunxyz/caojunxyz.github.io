U盘安装macOS踩坑记

2018-11-24
折腾

自从购买第一台苹果电脑(Macbook Air第一代）切换到苹果电脑办公已有6年历史，这期间重装系统的次数其实很少，总结一下大概有两种情况：1. 在公司接手一台别人用过的电脑，通过重装得到一个全新的系统。2. 用太久了，各种东西乱七八糟，通过重装整理。

安装盘制作

首先需要制作系统安装盘，可以通过命令行制作，也可以通过其它图形化工具制作。本文使用U盘在macOS系统中制作macOS Mojave安装盘为例。老款的Mac可能只支持USB2.0，可能不能识别3.0的U盘。首先在AppStore下载Mojave，下载完成后关闭安装提示界面，路径为/Applications/Install macOS Mojave.app。

命令行工具
  1. 通过磁盘工具格式化U盘，将U盘命名为*USB*（此处可取其它名字，下面命令中使用一致即可）

  1. 打开终端运行：sudo /Applications/Install\ macOS\ Mojave.app/Contents/Resources/createinstallmedia --volume /Volumes/USB

Etcher工具

​ Etcher是一个跨平台的开源工具，可以使用brew cask install etcher安装，非常方便。我之前也多次使用它制作各种启动盘，但不幸的是使用Etcher 1.4.6制作的Mojave安装盘无法识别，此处花费我大量时间，因为此前成功过就没有怀疑是它的问题。也许后续版本会修复问题，但也提了个醒，苹果官方的命令行安装方法是最可靠的。当使用其它方法遇到问题时可以快速用标准方法比较一下。使用方法很傻瓜，此处不啰嗦。此处要注意选择的镜像文件路径为/Applications/Install macOS Mojave.app/Contents/SharedSupport/InstallESD.dmg

另外的坑

老款Mac mini甚至连2K显示器都不支持，我连接到自己的4K显示器上发现没有信号，接着连到同事的2K显示器也没有信号，最后连到一个同事的1080P终于有信号了。开始也没有注意这个问题，因为公司很多个不同型号的Mac mini。

从网络安装

在开机时通过不同的组合按键可以通过网络安装不同的版本，如下所示：

Command (⌘)-R 安装您的 Mac 上装有的最新 macOS

Option-⌘-R 升级到与您的 Mac 兼容的最新 macOS

Shift-Option-⌘-R 安装 Mac 随附的 macOS 或与它最接近且仍在提供的版本

鉴于国内的网络状况，从网络安装失败的概率很大。我曾经电话咨询过苹果技术支持，他们也建议通过U盘安装……
